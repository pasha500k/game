# Elarian Skies

""Эларианские Небеса" — атмосферное приключение в открытом мире, созданное специально для этого тестового репозитория. Вас ждёт процедурно сгенерированный мир Эларии, сюжет с несколькими этапами, консоль разработчика и ориентировочно час игрового процесса, если исследовать мир и выполнять задания последовательно.

## Возможности

- 🌍 **Открытый мир** — более 6 000 тайлов, разные биомы (леса, горы, побережья, деревни) и точки интереса.
- 🌅 **Смена суток** — динамическое освещение и сумеречные оттенки.
- 📜 **Сюжет и журнал** — три сюжетные главы, интерактивные NPC и журнал заданий.
- 🧭 **Пять маяков Люмита** — исследуйте мир и активируйте древние маяки, чтобы наполнить артефакт энергией.
- 🧑‍🤝‍🧑 **NPC и диалоги** — несколько персонажей с уникальными репликами.
- 💻 **Консоль команд** — встроенная консоль (клавиша `\``) с командами: `help`, `teleport`, `time`, `give`, `heal`, `quest`, `whereami`, `spawn`.
- 🧭 **Миникарта и HUD** — индикаторы здоровья и выносливости, часы, миникарта, всплывающие подсказки сюжета.

## Установка

```bash
pip install -r requirements.txt
```

> **Примечание:** Pygame требует доступ к отображению окна. В среде без GUI игра не запустится.

## Запуск

```bash
python -m src.main
```

## Управление

| Действие | Клавиша |
|----------|---------|
| Перемещение | WASD или стрелки |
| Спринт | Shift |
| Взаимодействие / диалог | Пробел |
| Открыть/закрыть журнал заданий | Tab |
| Открыть консоль | ` (тильда) |
| Выход | Esc |
| Полноэкранный режим | F11 |

## Структура проекта

```
src/
├── console.py   # консоль команд
├── entities.py  # игрок, NPC, диалоги
├── main.py      # основной игровой цикл
├── settings.py  # настройки
├── story.py     # сюжет и прогрессия
├── ui.py        # HUD, миникарта, интерфейс
└── world.py     # генерация и отрисовка мира
```

## Советы по прохождению

1. Поговорите с Мудрой Арией в стартовой деревне.
2. Используйте миникарту и компас (яркие круги на карте), чтобы находить маяки.
3. Каждый активированный маяк добавляет запись в журнал и усиливает артефакт.
4. После активации всех пяти маяков вернитесь к Арии, чтобы завершить историю.
5. Консольные команды помогут исследовать мир быстрее или повторно просмотреть журнал.

Приятного путешествия по Эларии!
