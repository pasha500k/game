# Elarian Skies: Lumite Horizon

«Эларианские Небеса: Горизонт Люмита» — трёхмерное приключение в открытом мире, созданное на базе движка [Ursina](https://www.ursinaengine.org/). Игрок попадает на парящий остров, где нужно исследовать ландшафты, встретиться с наставниками, активировать шесть кристаллических маяков и вернуть свет в небеса. Игра рассчитана на неспешное прохождение (~1 час) с исследованием, прыжками по рельефу и использованием консоли разработчика.

## Ключевые особенности

- 🌄 **Полноценное 3D** — процедурно сгенерированный остров с рельефом, водой, деревьями и динамическим небом.
- 🧍 **Свободное перемещение** — поддержка WASD, прыжков, спринта и взаимодействия с NPC в трёхмерном пространстве.
- 🕰️ **Смена суток** — плавный цикл день/ночь, меняющий цвет неба, освещение и атмосферу.
- 🧭 **Сюжет и компас** — встречи с Арией, Рином и Лио, журнал заданий и компас, указывающий на ближайший маяк.
- 💬 **Встроенная консоль** — команды `help`, `heal`, `give`, `teleport`, `time`, `whereami`, `quest`, `spawn` помогают экспериментировать и отлаживать игровой процесс.

## Установка

```bash
pip install -r requirements.txt
```

> **Важно:** Для запуска необходима настольная ОС с поддержкой OpenGL. В безграфических средах игра не стартует.

## Запуск

```bash
python -m src.main
```

## Управление

| Действие | Клавиша |
|----------|---------|
| Перемещение | WASD |
| Прыжок | Пробел |
| Спринт | Shift |
| Взаимодействие с NPC | E |
| Открыть/закрыть консоль | ` (тильда) |
| Полноэкранный режим | F11 |
| Выход из игры | Esc |

## Консольные команды

| Команда | Описание |
|---------|----------|
| `help` | Список доступных команд |
| `heal [amount]` | Восстанавливает здоровье |
| `give <item> [count]` | Добавляет предмет в инвентарь |
| `teleport <x> <z>` | Перемещает к указанным координатам (высота подбирается автоматически) |
| `time <hours[:minutes]>` | Устанавливает время суток |
| `whereami` | Показывает текущие координаты |
| `quest` | Отображает активную цель и прогресс по маякам |
| `spawn <name>` | Призывает нового спутника рядом с игроком |

## Советы

1. Осмотритесь в стартовом лагере и поговорите с Арией, чтобы получить компас и первую цель.
2. Спринт расходует выносливость — планируйте рывки при подъёме на холмы и прыжках.
3. Компас внизу экрана показывает направление до ближайшего неактивированного маяка и расстояние до него.
4. После активации всех маяков вернитесь к Арии, чтобы завершить историю и увидеть финальное событие.
5. Не бойтесь экспериментировать с консолью — команды помогают исследовать ландшафт и тестировать механику.

Приятного исследования парящих островов Эларии!
